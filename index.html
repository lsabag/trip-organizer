טיולים מובילים השבוע

הגדרת גוגל מאפס
כדי להציג מפת גוגל עם ציוני ביקורות יש צורך במפתח API של Google Maps.
המפתח נדרש לשימוש ב-Maps JavaScript API ו-Places API.
המפתח נשמר בדפדפן בלבד ולא נשלח לאף שרת.
איך מקבלים מפתח?
1. גשו ל־console.cloud.google.com
2. צרו פרויקט חדש (חינמי)
3. הפעילו את Maps JavaScript API ו-Places API
4. תחת "Credentials" – צרו API Key
5. הדביקו כאן
